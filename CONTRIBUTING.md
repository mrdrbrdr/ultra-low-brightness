# Contributing

Thanks for wanting to improve this project!

## How to Contribute

1. **Found a bug?** Open an issue describing what went wrong
2. **Have an idea?** Open an issue to discuss it first
3. **Want to fix something?** Fork the repo, make your changes, and submit a PR -- I'm always up for collaboration!

## Testing Your Changes

Before submitting:
- Test on your own system
- Make sure the brightness control still works correctly
- Check both the 0-5% range and 5-100% range

## Pull Request Guidelines

- Describe what your change does and why (ie. what system youre using, etc)
- Reference any related issues
- Keep changes focused (one feature/fix per PR)

## Questions?

Open an issue or reach out! This is a small project and I'm happy to help.

## Code Style

Lets keep it readable and comment non-obvious parts.
